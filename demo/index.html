How to annotate your Image Classification
Dataset at minimum cost?


This website is based on the paper “Towards Good Practices for Efficiently Annotating Large-Scale Image Classification Datasets” accepted by CVPR21 as Oral presentation. For more details, check here and here.

In this project, we walk you through some of the main steps in annotating a image classification dataset as a task designer who cares about the money it costs. We starts from a given unlabeled dataset and a set of classes of interests. You are allowed to choose which kind of workers to hire.
